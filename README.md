# Traveler's hub <a name="about-project"></a>

**Traveler's Hub** "Traveler's Hub" is a web application for a company that provides commercial and scientific space travel services. The application will allow users to book rockets and join selected space missions.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

This project was built using these technologies.

- React
- Redux

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

This project requires the following:

- Node.js
- React

### Setup

Clone this repository to your desired folder:

```sh
  git clone https://github.com/LeslieAine/travelers-hub.git
  cd travelers-hub
```

### Install

Install this project with:

```sh
  npm install
```

### Usage

To run the project, execute the following command:

```sh
    npm start
```

### Run tests

To run tests, run the following command:

```sh
    npm test
```
